Portal Rural El Progreso

Descripción

Este proyecto es una página web sencilla hecha para mostrar información de la vereda El Progreso. En la página se puede ver información sobre agricultura y ganadería de la comunidad, y también tiene un formulario para que las personas se puedan inscribir a talleres rurales.

Lenguajes usados

Para hacer esta página se utilizaron:

 HTML para la estructura de la página
 CSS para el diseño
 JavaScript para validar el formulario

Estructura de la página

La página utiliza etiquetas semánticas para organizar mejor el contenido como:

header para el encabezado
nav para el menú de navegación
main para el contenido principal
section para dividir la información
article para mostrar información específica
footer para la información final

Funcionalidades

La página tiene un formulario de inscripción donde se validan algunos datos con JavaScript:

El nombre solo permite letras
La edad debe ser mayor a 12 años
El número de contacto debe tener exactamente 10 dígitos

Si los datos están correctos aparece un mensaje indicando que la inscripción fue realizada correctamente.

Pruebas

Se hicieron pruebas escribiendo datos correctos e incorrectos para verificar que el formulario funcionara bien. También se revisó la consola del navegador para comprobar que no hubiera errores.
